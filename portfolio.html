NFL, NBA rankings

  This was an entertaining project I completed alone. The idea is sports teams can be ranked solely based on the level of competition they beat. Using the rankings, a prediction can be made if one team will beat another in the future.

  I used the PageRank algorithm for this project where each node starts with the same value but divides that value equally among the nodes it points to. Thus, a highly ranked node is one that is pointed to from nodes that do not point to many nodes. Using this system, a team is a node that points to a team that it has lost to. With this logic, the best team would be the team with wins over teams with few losses. To gather data from thousands of games I used a python package called BeautifulSoup for web scraping. Implementing the algorithm consists of creating an nxn matrix for n teams representing the graph of nodes and edges pointing to other nodes. After a series of matrix multiplications, the values converge then the sum of a respective team column is that teams value.

  Using the algorithm continuously with the NBA season was entertaining to see how big upsets can rearrange the rankings. I’ve also implemented this algorithm with a previous year’s NFL regular season to rank the teams then tested its accuracy predicting the winner for the playoffs. Out of the 13 games played in the playoffs, the team with the higher ranking won 11 games.

  This is just one an example of the algorithm working but it is clear for an accurate prediction there would need to be more inputs such as player injuries and game location. Overall, this was a fun way to learn web scraping, a new algorithm, and get some experience cleaning, visualizing, and analyzing data.
